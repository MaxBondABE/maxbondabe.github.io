sqigl Documentation

Write SQL fluently

  • sqigl (pronounced "squiggle") is a build tool for SQL.
  • Traditional tools constrain you to writing SQL as a linear series of migrations.
  • sqigl allow you to write SQL like you would any other language - as a file tree organized by topic.
